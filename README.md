 FarmStack – Agriculture Intelligence Platform

FarmStack is a lightweight agriculture intelligence platform designed to help farmers access critical farming insights in a single place.
The platform integrates AI-based plant disease detection, market price tracking, weather advisories, and government schemes information.

The project focuses on edge AI deployment, where machine learning models run directly in the browser using TensorFlow.js, enabling real-time plant disease detection without requiring a backend server.

 Features
1. AI Plant Doctor

Farmers can upload an image of a diseased plant leaf and the system will detect the disease instantly using a machine learning model running directly in the browser.

Key capabilities:

Client-side AI inference

TensorFlow.js model deployment

Instant prediction with no server latency

Works even with limited connectivity

2. Market Price Dashboard

Displays crop market prices to help farmers make informed selling decisions.

Potential integrations:

Government mandi APIs

Local agricultural market data

Real-time commodity trends

3. Weather Advisory

Provides weather insights relevant to agriculture.

Possible features:

Rain alerts

Temperature forecasts

Humidity data

Crop advisory recommendations

Weather data can be integrated using APIs such as:

Open-Meteo

OpenWeatherMap

4. Government Schemes Section

A centralized section where farmers can learn about agricultural schemes such as:

PM-KISAN

Crop insurance schemes

Subsidies

Fertilizer support programs

5. Knowledge Library

An information hub containing:

Crop management practices

Pest prevention guides

Soil health knowledge

Agricultural best practices
